# Homebrew Tap for LeanMCP

This is the official Homebrew tap for LeanMCP CLI.

## Installation

```bash
brew tap rosaboyle/leanmcp
brew install leanmcp
```

## Usage

```bash
leanmcp --help
leanmcp version
leanmcp projects list
leanmcp chats list
```
