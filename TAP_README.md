# Homebrew Tap for MCLI

This is the official Homebrew tap for MCLI (LeanMCP CLI).

## Installation

```bash
brew tap rosaboyle/mcli
brew install mcli
```

## Usage

```bash
mcli --help
mcli version
mcli projects list
mcli chats list
```
